Quick Start — cURL
# Analyze text for bias
curl -X POST https://api.fairlens.dev/api/v1/analyze/text \
  -H "X-API-Key: fl_live_your_key" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "We are looking for a young, aggressive salesman...",
    "categories": ["gender", "age"]
  }'

# Response
{
  "overall_score": 34.2,
  "bias_scores": [
    {"category": "gender", "score": 28.5, "flagged_phrases": ["salesman"]},
    {"category": "age", "score": 40.0, "flagged_phrases": ["young"]}
  ],
  "recommendations": [
    "Replace 'salesman' with 'salesperson' or 'sales representative'",
    "Remove age-specific language like 'young'"
  ]
}
