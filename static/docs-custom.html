Analysis Endpoints — /api/v1/analyze
POST /api/v1/analyze/text All Tiers
Analyze text for bias across gender, race, age, and disability categories. Returns per-category scores (0-100), flagged phrases with suggestions, and overall fairness score.
Example Request & Response
{ "text": "We need a young, aggressive salesman...", "categories": ["gender", "race", "age", "disability"] } // Response { "report_id": "abc-123", "overall_score": 34.2, "bias_scores": [ {"category": "gender", "score": 28.5, "flagged_phrases": [...]}, {"category": "age", "score": 40.0, "flagged_phrases": [...]} ], "recommendations": ["Replace 'salesman' with 'salesperson'"] }
POST /api/v1/analyze/dataset All Tiers
Analyze a dataset for fairness metrics: disparate impact ratio, statistical parity difference, equal opportunity difference. Pass data as a list of JSON objects.
GET /api/v1/analyze/report/{report_id} All Tiers
Retrieve a previously generated analysis report by ID.
Fairness Endpoints — /api/v1/fairness
POST /api/v1/fairness/audit All Tiers
Racial fairness audit on JSON data. Returns disparate impact ratios per group, findings, and recommendations against EEOC 4/5ths rule or community-defined threshold.
POST /api/v1/fairness/audit/csv Pro+
Upload a CSV file for racial fairness audit.
POST /api/v1/fairness/audit/pdf Pro+
Audit CSV data and return a formatted PDF compliance report.
POST /api/v1/fairness/audit/remediate Pro+
Full remediation loop: audit → reweight → post-audit comparison.
POST /api/v1/fairness/audit/debias Enterprise
Adversarial ML debiasing via Fairlearn ExponentiatedGradient with DemographicParity constraint.
POST /api/v1/fairness/reweight All Tiers
Fairness-aware sample reweighting. Returns original and reweighted outcome rates per group.
POST /api/v1/fairness/reweight/csv Pro+
Upload a CSV file for fairness-aware reweighting.
Compliance Endpoints
NYC LL144 Michigan HB 4668 Colorado AI Act EEOC 4/5ths
POST /api/v1/fairness/audit/compliance Pro+
Check dataset compliance against a community-defined fairness standard (CDF v1.0).
POST /api/v1/fairness/audit/compliance/{regulation} Enterprise
Generate a regulation-specific compliance report. Supported regulations: ll144, michigan_hb4668, colorado_ai_act
Example Request
curl -X POST /api/v1/fairness/audit/compliance/ll144 \ -H "X-API-Key: fl_live_your_key" \ -F "file=@hiring_data.csv" \ -F "race_col=race" \ -F "outcome_col=hired" \ -F "favorable_value=yes" \ -F "config_json={}"
Community Governance — CDF v1.0
POST /api/v1/fairness/community/config Enterprise
Create a community fairness config with full provenance: priority groups, threshold, input protocol, participants, facilitator, location, date.
GET /api/v1/fairness/community/config Pro+
Retrieve the current active community fairness config for your account.
POST /api/v1/fairness/community/config/validate All Tiers
Validate a community config for completeness, staleness, and participant thresholds.
Keys & Billing
API KEY MANAGEMENT
POST /api/v1/keys/register Public
Register a new user and generate your first API key. No auth required.
POST /api/v1/keys Authenticated
Create additional API keys for your account (max 10 per user).
GET /api/v1/keys Authenticated
List all API keys for your account.
DELETE /api/v1/keys/{key_id} Authenticated
Revoke an API key. Cannot revoke your last active key.
